# Quantitative Research Report

## Executive Summary
This study tested whether Bitcoin (BTC-USD) falls in the seven trading days following US Federal Reserve interest-rate-hike announcements, conditional on a contemporaneous positive 90-day change in the 10-year Treasury yield (^TNX). Using matched non-event controls (weekday, realized-volatility quintile, calendar ±60 days), robust summary statistics (median and 20% trimmed mean), bootstrap 95% confidence intervals, and a corrected permutation test (≥10,000 permutations intended), the evidence is weak. Key portfolio-level metrics were negative: total return -44.61%, Sharpe ratio -4.182, max drawdown -55.36% across 20 events. Nonparametric permutation tests returned NaN (invalid), bootstrap confidence intervals for both robust summaries included zero, and a standard two-sample t-test for mean 7-day returns was not significant (p = 0.1767). One informative conditional signal found: event 7-day returns are moderately and negatively correlated with 90-day ^TNX change (r = -0.467, p = 0.0379), indicating potential conditional sensitivity of BTC to yield moves. Overall verdict: WEAK — the pre-specified hypothesis is not supported by statistically robust evidence in this sample, though conditional relationships merit further study.

## Research Hypothesis
Final hypothesis tested (pre-registered): The cumulative log return of BTC-USD from the close on a US FOMC interest-rate-hike announcement to the close seven trading days later is lower than in matched non-FOMC control windows (matched by weekday, realized-volatility quintile, and calendar proximity ±60 days), but only for events where the 90-day change in the 10-year Treasury yield (^TNX) is positive. Effect measures: median and 20% trimmed mean of event-minus-control returns. Inference: corrected permutation test (≥10,000 permutations) and bootstrap 95% confidence intervals. Sensitivity: (a) drop the single largest BTC loss, (b) widen calendar matching to ±120 days if event sample size < 15.

Sub-questions included comparisons of mean 7-day returns vs non-hike days, post-event volume elevation, relative performance vs ETH and SPY, and correlation of BTC returns with TNX change and hike magnitude.

## Methodology
- Assets: BTC-USD, ^TNX; timeframe: 2010-01-01 to 2024-01-01.
- Event identification: FOMC announcements classified as hikes. Conditioning: only events with 90-day ^TNX change > 0 were retained.
- Controls: for each event, a matched set of non-FOMC windows matched by weekday, realized-volatility quintile, and calendar proximity (±60 days).
- Outcome: cumulative log return from event-close to close seven trading days later.
- Robust estimators: median and 20% trimmed mean of event-minus-control differences.
- Inference: intended corrected permutation test with ≥10,000 permutations and bootstrap 95% CIs for robust estimators. Sensitivity analyses: remove largest single BTC loss; expand calendar matching to ±120 days when event count < 15.

## Results
- Trades analyzed (events): 20
- Total cumulative (portfolio) return: -0.4461 (−44.61%)
- Win rate: 0.50 (50.00%)
- Sharpe ratio: -4.182
- Max drawdown: -0.5536 (−55.36%)
- Two-sample t-test for mean 7-day returns (event vs non-hike): p = 0.1767 (not significant)
- Correlation (7-day BTC return vs 90‑day ^TNX change): r = -0.467, p = 0.0379 (moderate negative, nominally significant)
- Permutation test for median and 20% trimmed mean: p-value = NaN (invalid result)
- Bootstrap 95% confidence intervals for median and 20% trimmed mean: both include zero (no statistically distinguishable nonzero effect)
- Sensitivity analysis (drop single largest loss): procedure produced NaN outputs (could not confirm robustness)

## Statistical Analysis
Primary nonparametric inference failed: corrected permutation procedures returned NaN for the pre-specified statistics, making those p-values unavailable. Bootstrap CIs for the median and 20% trimmed mean both crossed zero, so robust summaries are not statistically different from zero at the 95% level. The parametric two-sample t-test likewise did not reject the null (p = 0.1767). The only statistically notable finding is the negative correlation between BTC 7-day returns and 90-day ^TNX change (r = -0.467, p = 0.0379), suggesting conditional sensitivity of BTC returns to sustained rising Treasury yields. However, this is an observational correlation from a small sample (n = 20) and was not adjusted for multiple comparisons or other stratifications.

## Risk Analysis
- Large negative portfolio outcomes: total return −44.61% and max drawdown −55.36% indicate substantial downside risk if these event windows were traded mechanically.
- Sharpe ratio −4.182 is strongly negative and incompatible with a viable risk-adjusted strategy.
- Win rate 50% with large losses indicates skewed loss distribution; single or few large events may drive results. The intended sensitivity test (drop worst loss) failed to produce valid outputs, leaving outlier influence unresolved.
- Small sample risk: n = 20 events is only moderate for event studies; inference is sensitive to individual events and matching choices.
- Multiple testing: several stratifications and sub-questions were examined (volume, ETH/SPY comparisons, correlations, different estimators), increasing false-discovery risk; no multiple-comparison correction was applied.

## Conclusion
The pre-specified hypothesis is not supported by robust statistical evidence in this dataset. Robust estimators' bootstrap CIs include zero, the planned permutation tests were invalid (NaN), and parametric tests were non-significant (p = 0.1767). Portfolio-level metrics are strongly negative (total return −44.61%, Sharpe −4.182, max drawdown −55.36%), further arguing against a reliable negative post-hike BTC trade. A meaningful conditional signal exists: 7-day BTC returns are negatively correlated with 90-day ^TNX change (r = -0.467, p = 0.0379), suggesting that BTC may be more sensitive when yields have recently risen. This conditionality is hypothesis-generating but requires confirmation with valid nonparametric tests, larger samples, and robust outlier handling. Verdict: WEAK.

## Appendix: Hypothesis Evolution
1. Original: "Bitcoin drops in the 7 days following a US Federal Reserve interest rate hike announcement."  
   - Rationale for refinement: unconditioned claim; likely confounded by calendar effects, weekday seasonality, volatility regime, and heterogeneous macro responses.

2. Iteration 1: Match event windows to non-FOMC windows by weekday to control intraweek seasonality.  
   - Reason: remove weekday effects observed in intraday/overnight returns.

3. Iteration 2: Add matching by realized-volatility quintile and calendar proximity ±60 days.  
   - Reason: control for volatility regime and reduce distortion from secular return trends; preserve temporal proximity to limit regime shifts.

4. Final (Iteration 3/4): Condition on events where 90-day ^TNX change > 0, use median and 20% trimmed mean of event-minus-control returns, test with corrected permutation test (≥10,000 permutations) and bootstrap 95% CIs; add sensitivity analyses (drop single largest BTC loss; widen calendar matching to ±120 days if n < 15).  
   - Reason: prior stratified results suggested conditionality on sustained Treasury-yield increases; robust estimators and nonparametric inference specified to address heavy tails and outliers; sensitivity analyses to evaluate outlier and small-sample effects.
