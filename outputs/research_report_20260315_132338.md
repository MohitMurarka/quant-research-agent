# Quantitative Research Report

## Executive Summary
We tested the claim that the S&P 500 produces strictly positive total returns over every rolling 10‑year period, using adjusted‑close (total‑return) ETFs as proxies: VOO, IVV and RSP over the sample window 1990‑01‑01 to 2024‑01‑01 (instrument-level data availability varies). The literal universal claim is falsified: at least one observed 10‑year rolling window for VOO produced a small negative cumulative total return (minimum = -0.0027, i.e., −0.27%). Nevertheless, the empirical evidence shows that positive 10‑year total returns are overwhelmingly likely historically: IVV had 98.27% positive 10‑year windows (bootstrap 95% CI [97.91%, 98.59%]) and RSP had an unconditional 99.92% positive rate (conditional bottom‑decile prior performance windows for RSP were 100.00% positive, n=495). Statistical tests (bootstrap CIs and a permutation test) are reported and support the statement that "positive 10‑year outcomes are very likely" while rejecting the strict "always positive" claim. Key portfolio-level metrics from the backtest: Total Return 454.71%, Annualized Return 13.76%, Sharpe Ratio 0.601, Max Drawdown −33.99%.

## Research Hypothesis
Original / Final Hypothesis Tested: "S&P500 has positive returns over any rolling 10‑year period." Operationalized as: for each trading day in the sample, compute the 10‑year cumulative total return (using adjusted close) and test whether all such daily windows are > 0. Additional sub‑questions examined the fraction of positive windows and conditional behavior by recent prior performance.

## Methodology
- Assets/proxies: VOO, IVV, RSP (adjusted‑close, total‑return prices).
- Timeframe and data availability: analysis nominally covers 1990‑01‑01 to 2024‑01‑01, but ETF-level data spans differ: VOO 2010‑09‑09–2023‑12‑29 (3,350 rows), IVV 2000‑05‑19–2023‑12‑29 (5,941 rows), RSP 2003‑05‑01–2023‑12‑29 (5,203 rows).
- For each asset, compute daily rolling 10‑year cumulative total returns where available (10‑year = 10 calendar years from each start date).
- Sub‑analyses:
  - VOO: check whether the minimum 10‑year cumulative return > 0.
  - IVV: compute fraction of positive 10‑year windows and quantify uncertainty via a bootstrap 95% confidence interval and estimate probability that a bootstrap resample yields 100% positive.
  - RSP: compute unconditional fraction positive and conditional fraction positive restricting windows whose start date had the prior 12‑month simple return in the bottom decile. Test conditional vs unconditional using a one‑sided permutation test.

- Nonparametric inference methods used: bootstrap confidence interval for a proportion; permutation test for the conditional comparison. (Backtest output provides the exact bootstrap CI and permutation p‑value.)

## Results
- VOO: Number of 10‑year rolling windows = 3,349. Minimum 10‑year cumulative return = −0.0027. Conclusion: not all daily rolling 10‑year windows are positive (hypothesis falsified in the strict sense).
- IVV: Number of 10‑year rolling windows = 5,940. Fraction positive = 98.27%. Bootstrap 95% CI = [97.91%, 98.59%]. Bootstrap estimate of probability a resample is 100% positive = 0.000000. Conclusion: fraction is statistically distinguishable from 100% (CI excludes 100%).
- RSP (unconditional): total valid 10‑year windows with prior‑12m available = 4,949. Unconditional fraction positive = 99.92%. RSP conditional (start-date prior 12m in bottom decile): n = 495 windows, conditional fraction positive = 100.00%. Observed difference (unconditional − conditional) = −0.08%. Permutation test one‑sided p‑value = 1.000000. Conclusion: no evidence that conditional fraction is lower than unconditional.

Portfolio-level backtest metrics (aggregate/statistics reported): Total Return = 454.71%; Annualized Return = 13.76%; Sharpe Ratio = 0.601; Max Drawdown = −33.99%; Win Rate reported = 54.46% (see caveat below); Number of trades/windows analyzed = 3,349.

## Statistical Analysis
- IVV’s positive‑window proportion is estimated at 98.27% with a narrow bootstrap CI [97.91%, 98.59%], which excludes 100% and therefore rejects the universal hypothesis at conventional confidence levels.
- The bootstrap estimate that a resample yields 100% positive IVV windows is 0.000000 (i.e., effectively zero under the bootstrap procedure used).
- For RSP the unconditional positive fraction is 99.92%; the conditional sample (bottom‑decile prior 12m, n=495) produced 100% positive windows and the permutation test yields a one‑sided p‑value = 1.0 for the hypothesis that conditional < unconditional (no evidence of conditional degradation).
- The single negative VOO 10‑year window (−0.27%) is small in magnitude; however, a single counterexample is sufficient to falsify the literal universal claim.

## Risk Analysis
- Data span heterogeneity: ETF start dates differ (VOO data starts 2010), producing unequal effective sample lengths; direct comparisons should account for this.
- Metric ambiguity: "Win Rate = 54.46%" is inconsistent with the reported proportions of positive 10‑year windows and is undefined in the output; this reduces interpretability.
- Methodological/documentation gaps: the bootstrap/resampling parameters, Sharpe annualization method (risk‑free rate and return frequency), and data cleaning decisions (treatment of dividends, corporate actions) are not documented here and may affect numeric precision.
- Potential biases: survivorship, selection of ETF proxies, and structural/regime changes over the century may limit forward extrapolation. The single small negative window could reflect dividend timing, ETF inception idiosyncrasy, or real market behavior.
- Economic risk: even with high probability of positive 10‑year returns historically, realized drawdowns can be large (observed max drawdown −33.99%).

## Conclusion
The strict hypothesis "S&P500 has positive returns over any rolling 10‑year period" is not supported — it is falsified by an observed 10‑year window for VOO with cumulative return −0.0027. However, the data provide strong evidence that positive 10‑year total returns have been overwhelmingly likely historically: IVV positive windows = 98.27% (95% CI [97.91%, 98.59%]), RSP unconditional = 99.92%. Thus, while "always positive" is false, the practical conclusion is that 10‑year horizons have historically been very likely to produce positive total returns, albeit not guaranteed. Users should weigh sample limitations and documented caveats before applying these results in decision rules.
